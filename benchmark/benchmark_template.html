Methodology & caveats

Each record was extracted directly from its paper's PDF by an LLM agent following a shared rubric (benchmark/rubric.md), covering result metrics, model architecture, input/preprocessing, training data and regime, and a plain-language contribution summary. Fields not explicitly stated in a paper are marked not reported rather than inferred or guessed. A sample of records was cross-checked against source PDFs after extraction.

Two rank collisions exist in the source literature. CinC2020-134 and CinC2020-225 each self-report placing 16th of 41; CinC2020-185 and CinC2020-198 each self-report placing 24th of 41. Both papers in each pair state this explicitly in their own text — this benchmark preserves each paper's claim rather than arbitrating between them, since no authoritative per-team leaderboard table was recoverable from the challenge overview paper (the official scores live in an external results CSV, not the PDF).
Not every score is on the same footing. Several teams could not get a working submission scored on the official hidden test set and report only cross-validation or internal-validation numbers instead — these are captured under each record's self-reported metrics with official_rank left as not reported, rather than being mixed into the ranked comparison.

Two papers in the source set are not team entries and are excluded from the default view: 2020ChallengePaper.pdf and CinC2020-236.pdf are both organizer overview papers describing the Challenge itself (the latter a shorter companion write-up by the same authors). CinC2020-340.pdf is unrelated — a system built for the 2018 PhysioNet Challenge, published in the same proceedings volume.
